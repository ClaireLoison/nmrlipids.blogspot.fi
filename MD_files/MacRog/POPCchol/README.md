### Systems

POPC bilayer with various concentration of Cholesterol (0, 10, 40, 50, 60 %)

### How to run (Gromacs 4.6)

```
mdrun -deffnm md_cholXX &> mdrun.log
```
where XX denotes the concentration of cholesterol

### Retrieve data

All simulations are under the same dataset at Zenodo: http://dx.doi.org/10.5281/zenodo.13877
