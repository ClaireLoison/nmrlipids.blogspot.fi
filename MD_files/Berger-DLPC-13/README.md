## Berger-DLPC-13

### Forcefield

Berger force field delivered by Peter Tieleman (http://wcm.ucalgary.ca/tieleman/downloads) and modified for DLPC[1].
More information about the Berger-DLPC-13 forcefield can be found here[1].
