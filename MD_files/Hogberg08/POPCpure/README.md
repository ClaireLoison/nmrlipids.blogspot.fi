### System

128 molecules of POPC at full hydration without ions.

### Relevant files:

 - md.input        : main MD input file 
 - dmpc-2008.mmol  : lipid topology/force field file
 - H2O.mmol        :  SPC water

### How to run (MDynaMix v 5.2)

```
mdp < md.input
```
