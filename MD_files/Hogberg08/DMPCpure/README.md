### System

98 molecules of DMPC at full hydration without ions.

### Relevant files:

 - md.input        : main MD input file 
 - 160181_pc.mmol  : lipid topology/force field file
 - H2O.mmol        :  SPC water

### How to run (MDynaMix v 5.0)

```
mdp < md.input
```
